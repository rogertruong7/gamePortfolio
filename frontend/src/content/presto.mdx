A Google Slides clone called Presto, built as a React web application. Users can create accounts, build slideshow presentations with text, images, videos, and syntax-highlighted code blocks, then preview them in a presentation mode.

---

## Overview

Presto is a full-featured presentation editor inspired by Google Slides and slides.com. It provides a dashboard for managing presentations, a slide editor with drag-and-drop element placement, and a preview mode for presenting. Built with React and Material UI, deployed on Vercel with Vercel KV for persistence.

---

## Features

- **User Authentication**: register, login, and logout with session management
- **Presentation Dashboard**: create, rename, delete, and browse all your slide decks
- **Slide Editor**: add and arrange multiple slides with a visual editor
- **Text Blocks**: add and edit text elements with positioning and styling
- **Image Blocks**: insert images into slides with resize and reposition
- **Video Blocks**: embed videos directly into presentations
- **Code Blocks**: syntax-highlighted code blocks supporting C, Python, and JavaScript
- **Theme & Background Picker**: customise presentation themes and slide backgrounds
- **Presentation Preview**: full-screen presentation mode with keyboard arrow navigation
- **Element Positioning**: drag and resize elements freely on the slide canvas

---

## Tech Stack

- **Frontend**: React.js, Vite
- **UI Library**: Material UI (MUI)
- **Deployment**: Vercel
- **Database**: Vercel KV
- **Linting**: ESLint

## Getting Started

1. **Clone & install**:
   ```bash
   git clone https://github.com/rogertruong7/presto-deploy.git
   cd presto-deploy/frontend
   npm install
   ```

2. **Run the dev server**:
   ```bash
   npm run dev
   ```
